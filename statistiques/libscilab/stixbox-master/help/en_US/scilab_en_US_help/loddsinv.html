loddsinv

inverse of log odd

Calling Sequence

p = loddsinv(z,b)

Description

Compute the inverse of log odds. The function is equal to 1/(1+exp(-u)) with

u = z

if only one input argument is given

u = z*b

if size(z,2) = length(b)

u = z*b(1:n-1) + b(n)
